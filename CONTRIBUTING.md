# Contributing to pulse-sort

Contributions, issues, and feature requests are welcome.

## Getting started

```bash
git clone git@github.com:rrajmanna/pulse-sort.git
cd pulse-sort
python3 -m venv venv
source venv/bin/activate
pip install -e .[dev]
```

## Running tests

```bash
cd tests
python -m pytest -v
```

See `docs/api/` for documentation of each module in `src/`.
